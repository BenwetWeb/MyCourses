Урок 1. Частина перша

Нові слова. Слухайте, повторюйте, запам'ятовуйте:

  • der Bleistift [де́а бляйштифт] -> олівець
  • das Brot [дас брот] -> хліб
  • das Buch [дас бух] -> книга
  • das Heft [дас хефт] -> зошит
  • der Kuli [де́а кулі] -> ручка
  • die Kartoffeln [ді картофельн] -> картопля (мн.)
  • die Suppe [ді зуппе] -> суп
  • die Tasche [ді таше] -> сумка, кишеня
  • die Wurst [ді вурст] -> ковбаса
  • die Tafel [ді тафель] -> дошка

Інтерактивна вправа: Перевір свою пам'ять!

Урок 1. Частина перша (продовження)

Використання нових слів з артиклем та дієсловом "бути" (`ist`):

Прочитайте вголос німецькі речення та зверніть увагу на український переклад:

  • Das Buch und das Heft. -> Книга і зошит.
  • Dort ist der Bleistift. -> Там є олівець.
  • Die Tasche ist hier. -> Сумка тут.
  • Das Brot und die Wurst. -> Хліб і ковбаса.
  • Eine Suppe und Kartoffeln. -> Суп і картопля.
  • Das ist eine Tasche. -> Це є сумка.

Урок 2. Продовження

Повторюємо дієслова та вживаємо їх у реченнях:

Прочитайте вголос німецькі речення та зверніть увагу на український переклад:

  • Sie isst hier. -> Вона їсть тут.
  • Das ist ein Buch. -> Це є книга.
  • Sie isst dort. -> Вона їсть там.
  • Das ist ein Heft. -> Це є зошит.
  • Er isst hier. -> Він їсть тут.
  • Das ist ein Brot. -> Це є хліб.
  • Er isst dort. -> Він їсть там.
  • Das ist eine Wurst. -> Це є ковбаса.
  • Das ist ein Bleistift. -> Це є олівець.
  • Das ist eine Tafel. -> Це є дошка.
  • Das ist ein Kuli. -> Це є ручка.
  • Das ist eine Tasche. -> Це є сумка.

Важливо:

З попереднього уроку ми вже знаємо, що слова, які позначають іменники, пишуться з великої літери. В німецькій мові іменники мають рід (чоловічий, жіночий, середній), який позначається артиклем.

Вживання дієслова "брати/приносити" (`holt`) та неозначеного артикля:

  • Bitte, holt die Suppe! -> Будь ласка, принесіть суп!
  • Bitte, holt den Bleistift! -> Будь ласка, принесіть олівець!
  • Bitte, holt das Brot! -> Будь ласка, принесіть хліб!
  • Bitte, holt die Suppe! -> Будь ласка, принесіть суп!
  • Bitte, holt die Wurst! -> Будь ласка, принесіть ковбасу!
  • Bitte, holt die Kartoffeln! -> Будь ласка, принесіть картоплю!

Перекладіть на українську мову, вживаючи неозначений артикль:

  • Покажіть, одну порцію хліба. -> Bitte, ein Brot.
  • Покажіть, стакан. -> Bitte, ein Glas.
  • Покажіть, пляшку. -> Bitte, eine Flasche.
  • Покажіть, сигарети. -> Bitte, Zigaretten.
  • Покажіть, картоплю. -> Bitte, Kartoffeln.

Урок 2. Продовження (частина друга)

Повторюємо дієслова та прислівники. Читайте німецькі речення та звертайте увагу на український переклад:

  • Sie trinken Bier. -> Вони п'ють пиво.
  • Er trinkt Milch. -> Він п'є молоко.
  • Sie trinkt Tee. -> Вона п'є чай.
  • Wir trinken Tee. -> Ми п'ємо чай.
  • Was trinkst du? -> Що ти п'єш?
  • Herr und Fräulein Huber stehen hier. -> Пан і пані Хубер стоять тут.
  • Das Buch und das Heft liegen dort. -> Книга і зошит лежать там.
  • Sie stehen hier links. -> Вони стоять тут ліворуч.
  • Ich trinke ein Glas Bier. -> Я п'ю склянку пива.
  • Wir essen Suppe und Kartoffeln. -> Ми їмо суп і картоплю.
  • Die Schule und der Bahnhof liegen dort rechts. -> Школа і вокзал лежать там праворуч.

Зверніть увагу:

Дієслово `liegen` (лежати, знаходитися) змінюється залежно від місця.

Прочитайте вголос німецькі речення, звертаючи увагу на переклад:

  • Ich trinke Milch. -> Я п'ю молоко.
  • Ich trinke ein Glas Milch. -> Я п'ю склянку молока.
  • Du trinkst Tee. -> Ти п'єш чай.
  • Du trinkst eine Flasche Bier. -> Ти п'єш пляшку пива.
  • Er trinkt Wasser. -> Він п'є воду.
  • Er trinkt ein Glas Wasser. -> Він п'є склянку води.
  • Sie trinkt Kaffee. -> Вона п'є каву.
  • Sie trinkt eine Tasse Kaffee. -> Вона п'є чашку кави.
  • Wir essen Brot. -> Ми їмо хліб.
  • Wir essen Wurst. -> Ми їмо ковбасу.
  • Wir essen Suppe. -> Ми їмо суп.
  • Wir essen Kartoffeln. -> Ми їмо картоплю.
  • Wo ist die Tafel? -> Де дошка?
  • Wo ist der Kuli? -> Де ручка?
  • Wo liegt der Bleistift? -> Де лежить олівець?
  • Wo sind die Kartoffeln? -> Де картопля?

Урок 2. Продовження (частина третя)

Займенники та іменники. Прочитайте вголос німецькі речення, звертаючи увагу на переклад:

  • Ich esse Suppe. -> Я їм суп.
  • Wir essen Kartoffeln. -> Ми їмо картоплю.
  • Sie trinken dort. -> Вони п'ють там.
  • Der Kuli und der Bleistift liegen hier. -> Ручка і олівець лежать тут.
  • Wo steht die Frau? -> Де стоїть жінка?
  • Dort stehen sie nicht. -> Вони там не стоять.
  • Wo liegt die Tasche? -> Де лежить сумка?
  • Er trinkt Kaffee. -> Він п'є каву.
  • Das ist mein Buch. -> Це моя книга.
  • Das ist mein Kuli. -> Це моя ручка.

Перекладіть на українську мову:

  • Я їм хліб. -> Ich esse Brot.
  • Ти п'єш пиво. -> Du trinkst Bier.
  • Він п'є молоко. -> Er trinkt Milch.
  • Вона п'є воду. -> Sie trinkt Wasser.
  • Ми їмо ковбасу. -> Wir essen Wurst.
  • Вони їдять суп. -> Sie essen Suppe.
  • Де книга? -> Wo ist das Buch?
  • Де зошит? -> Wo ist das Heft?
  • Де ручка? -> Wo ist der Kuli?
  • Де олівець? -> Wo ist der Bleistift?
  • Де сумка? -> Wo ist die Tasche?
  • Де ковбаса? -> Wo ist die Wurst?
  • Де картопля? -> Wo sind die Kartoffeln?
